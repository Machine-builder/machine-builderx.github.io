Shitty School Clicker Game

Premise of this actually awful game :
Write completely useless essays to gain essay-writing skill to write more useless essays...?


Press the button that looks like a pencil and paper to write an essay.
Essays by themselves are useless, but you can mark your essay
to turn it into 'school points'. Click the button that looks like a
checklist to convert your essays into points. Points are used to
upgrade and buy new things. Scroll down to see what you can buy.
Before you play the game, use the keyboard shortcut Ctrl+F5
To hard-refresh the page, allowing any new game scripts to reload

Click the button below to save your current game whenever.
Once the save game button is pressed, the game data will
be stored for up to one year. When the page is reloaded,
any found game saves will automatically load.
