FRG V1 (discontinued)

Due to the discontinuance, the program in no longer available for download from this website.
This program was originally developed to create structures in Minecraft Bedrock Edition, when structures needed hundreds or even thousands of files to function correctly. The program was designed to generate all those files for the user, to speed up the process a lot. However, once the 1.15 beta was introduced, structures could be made using only 3 files - A 'feature', 'feature_rule', and an 'mcstructure'.
There were two versions of the FRG V1; V1.1 (which generated the required 100+ files for every structure)
The next version - V1.2 was developed for the 1.15 beta (and above). It was the first version of the FRG to export mcstructure files.
