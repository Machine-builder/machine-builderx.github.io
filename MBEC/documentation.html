Entity Commands

Event Commands

Commands can now be used within entity events
This can be achieved with the following setup 


        "minecraft:entity_spawned": {
            "randomize": [
                {
                    "weight": 5,
                    "execute": { "commands": [ "/say this is option 1" ] }
                },
                {
                    "weight": 5,
                    "execute": { "commands": [ "/say this is option 2" ] }
                }
            ]
        }

On-death Commands

Commands can also be executed on an entityâ€™s death.
Here is an example setup 


        {
            "format_version": "1.16.0",
            "minecraft:entity": {
                "on_death_commands": [
                    "/say I just died :("
                ]
                // ...
            }
        }

Commands can be specified to only run if the entity was killed by the player.
Here's how to set that up 



        {
            "format_version": "1.16.0",
            "minecraft:entity": {
                "on_death_commands": {
                    "player_killed": [
                        "/say I was killed by a player"
                    ]
                }
                // ...
            }
        }
